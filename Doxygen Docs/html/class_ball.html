This is the BoundsChecking function.

This checks the ball's position against the sides, top, and bottom of the screen. It also checks using the CollisionManager for collision against the paddle.

This is the frameStarted function.

This calls the translate function every frame to move the ball according to it's velocity.
It also calls the BoundsChecking function every frame.

Note
translate is adjusted by evt.timeSinceLastFrame to ensure consistent speed across frame rates.

◆ HitBottom()

void Ball::HitBottom ( )

This is the HitBottom function.

This is called when the BoundCheck determines the ball has reached the bottom of the screen. It reversed the ball's speed, moves the ball to a safe position to avoid repeat collisions, and decrements the ball's lives.

◆ ReboundPaddle()

void Ball::ReboundPaddle ( )

This is the ReboundPaddle function.

This is called when the CollisionManager determines that the ball has collided with the paddle. It reverses the ball's speed, moves the ball to a safe position to avoid repeat collisions, and increments the ball's score.

◆ ReboundSides()

void Ball::ReboundSides ( )

This is the ReboundSides function.

This is called when the ball reaches the max or minimum screen bounds on the x axis. It simply reverses the ball's x velocity.
