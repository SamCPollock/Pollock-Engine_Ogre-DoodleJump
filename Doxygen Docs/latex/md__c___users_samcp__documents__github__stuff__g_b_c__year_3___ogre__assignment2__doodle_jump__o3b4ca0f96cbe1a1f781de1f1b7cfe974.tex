0. To build SDL, your machine must, at a minimum, have the Direct\+X9.\+0c SDK installed. It may or may not be retrievable from the \href{https://www.microsoft.com}{\texttt{ Microsoft}} website, so you might need to locate it \href{https://duckduckgo.com/?q=directx9.0c+sdk+download&t=h_&ia=web}{\texttt{ online}}. {\itshape Editor\textquotesingle{}s note\+: I\textquotesingle{}ve been able to successfully build SDL using Visual Studio 2019 {\bfseries{without}} the DX9.\+0c SDK}


\begin{DoxyEnumerate}
\item Open the Visual Studio solution file at {\ttfamily ./\+Visual\+C/\+SDL.sln}.
\item Your IDE will likely prompt you to upgrade this solution file to whatever later version of the IDE you\textquotesingle{}re using. In the {\ttfamily Retarget Projects} dialog, all of the affected project files should be checked allowing you to use the latest {\ttfamily Windows SDK Version} you have installed, along with the {\ttfamily \mbox{\hyperlink{class_platform}{Platform}} Toolset}.
\end{DoxyEnumerate}

If you choose {\itshape NOT} to upgrade to use the latest {\ttfamily Windows SDK Version} or {\ttfamily \mbox{\hyperlink{class_platform}{Platform}} Toolset}, then you\textquotesingle{}ll need the {\ttfamily Visual Studio 2010 \mbox{\hyperlink{class_platform}{Platform}} Toolset}.


\begin{DoxyEnumerate}
\item Build the {\ttfamily .dll} and {\ttfamily .lib} files by right clicking on each project in turn (Projects are listed in the {\itshape Workspace} panel in the {\itshape File\+View} tab), and selecting {\ttfamily Build}.
\end{DoxyEnumerate}

You may get a few warnings, but you should not get any errors.

\item Create a C++ file for your project.
\item Set the C runtime to {\ttfamily Multi-\/threaded DLL} in the menu\+: {\ttfamily Project$\vert$\+Settings$\vert$\+C/\+C++ tab$\vert$\+Code Generation$\vert$\+Runtime Library}.

{\itshape VC7 Specific\+: Instead of doing this, I find it easier to add the include and library directories to the list that VC7 keeps. The \char`\"{}include directory\char`\"{} I am referring to is the {\ttfamily ./include} folder.

Now we\textquotesingle{}re going to use the files that we had created earlier in the {\itshape Build SDL} step.

Copy the following file into your Project directory\+:


\begin{DoxyItemize}
\item {\ttfamily SDL2.\+dll}
\end{DoxyItemize}

Add the following files to your project (It is not necessary to copy them to your project directory)\+:


\begin{DoxyItemize}
\item {\ttfamily SDL2.\+lib}
\item {\ttfamily SDL2main.\+lib}
\end{DoxyItemize}

To add them to your project, right click on your project, and select {\ttfamily Add files to project}.
